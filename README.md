## About

ThermIQ is a responsive dashboard that visualizes device health with a clean Material 3 interface, smooth animations and reusable components.

This project started as a challenge:

- 🕒 Started: **3:00 AM**
- 🚀 Finished: **5:00 AM**
- 📅 Date: **10 June 2026**

I installed Flutter for the first time and challenged myself to learn the framework and ship a complete application before sunrise.
